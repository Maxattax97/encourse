get_test_summary.jsonify(test_data, hidden)[source]¶

Counts the number of students who have passed each test case

Sums up the total number of students who have passed each particular test case, and stores that information with the test case name. If the test case is hidden, “H” is appended to the end of the test case name

Args:

data (dict): A dictionary of users mapped to their respective test scores The dictionary has the following format:

{
    "name1": {
       "tests": {
            "Test1": (bool, int),
            ...
       },
       "total": int (percentage)
    },
    ...
}

hidden (bool): A boolean that is true if the test cases are hidden

Returns:

json: A json array containing the percentage of students who passed each test. The json has the form:

[
    {
        "testName": str,
        "hidden": bool,
        "score": int (percentage)
    },
    ...
]
