get_class_progress.jsonify(visible_data, hidden_data)[source]¶

Formats data for /classProgress endpoint

Converts information in data into an appropriately formatted json for the /classProgress endpoint

Args:

data (dict): A dictionary of the following format:

{
   "name1": {
        "Test1": (bool, float),
        ...
   },
   ...
}
Return:

dict: A dictionary of histogram data split into 20% bins:

{
    "0-20%": int,
    "20-40%": int,
    "40-60%": int,
    "60-80%": int,
    "80-100%": int
}

where each percentage bin contains a value 0 <= int <= 100
